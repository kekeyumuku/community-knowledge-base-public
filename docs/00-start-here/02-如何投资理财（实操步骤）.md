# 如何投资理财（实操步骤）

**重点：普通人开始投资之前，最重要的不是先找收益最高的地方，而是先把应急金、预算边界和资金分层建立起来。**

这篇文章解决一个具体问题：**参与 Web3 之前，你的钱应该怎么分配？**

它不讲 Web3 的风险属性（那部分在《Web3 投资注意事项》），只讲投资前的基础财务结构。

## 一、开始投资前，先做 4 件事

### 1. 给自己留好应急钱
应急金是投资之前的第一步。

它的意义很简单：
- 遇到失业、生病、家庭突发情况时，不至于被迫卖出资产
- 有安全垫后，面对市场波动时不容易慌
- 可以避免“小事故”升级成真正的财务问题

一般建议至少预留 **3-6 个月生活支出**。
如果工作不稳定，可以准备 **6-12 个月**。

### 2. 先算清楚自己能拿多少钱投资
这一步非常关键。

投资一定要用 **输得起的钱**，而不是影响正常生活的钱。

最简单的做法是：
- 先估算自己一年的收入
- 再估算自己一年的支出
- 收入减去支出后，看还有多少结余
- 再从结余里拿出一部分，作为可投资资金

如果你现在本来就没有结余，甚至还在透支、负债，那最优先的事情不是投资，而是先改善现金流。

### 3. 给投资做好防火墙
在开始进攻前，先做好保护。

这层“防火墙”通常包括：
- 基础保险
- 基本生活保障
- 资金分层

它的意义不是让你变保守，而是防止一次突发事件把你所有积累打回原点。

### 4. 明确资产分层
你可以把自己的资金简单分成 4 层：

1. **应急金账户**
2. **保障类支出账户**
3. **稳健型账户**
4. **高风险投资账户**

Web3、数字资产这一类，通常应该放在第四层，而不是第一层或全部身家。

## 二、普通人最容易犯的错误

### 错误 1：还没建立安全垫，就急着投资
这样一旦生活出现波动，你就会在最不该卖的时候被迫卖掉资产。

### 错误 2：拿影响生活的钱去投资
只要这笔钱一跌你就睡不着，那它大概率就不该拿来做高风险投资。

### 错误 3：把高风险投资当成翻身工具
这类投资更适合作为整体资产配置中的一小部分，而不是人生唯一出口。

## 三、如果你准备参与 Web3，资金放在哪一层？

如果你已经完成前面的准备，Web3 属于第四层：

> **高风险投资账户中的一小部分实验仓位。**

- 不是你的全部投资
- 不是你的生活保障
- 建立在前面三层都稳住之后再参与

关于 Web3 本身的风险属性和边界规则，见《Web3 投资注意事项》。

## 四、一个最实用的起点

如果你现在还很模糊，可以先用这个顺序：

1. 留出应急金
2. 算清楚年度收支和结余
3. 明确自己最多拿多少钱参与高风险投资
4. 再开始学习 Web3 的注册、买入和钱包

## 一句话总结

**投资真正的起点，不是找机会，而是先让自己在犯错时也不至于出局。**
