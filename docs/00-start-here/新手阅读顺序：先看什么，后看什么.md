# 新手阅读顺序：先看什么，后看什么

如果你是第一次接触 Web3，建议不要一上来就急着注册、买币、装钱包。
先把边界、风险和基本路径看清楚，再开始操作，会更稳。

这套知识库不是为了让你“冲进去”，而是为了帮你用**更低犯错率**的方式，完成从 0 到 1 的入门。

## 一、这套知识库适合谁

适合这几类人：

- 想小额了解 Web3、而不是一上来重仓的人
- 想先建立基础认知，再开始实操的人
- 想尽量少踩坑、少犯错的人
- 想把 Web3 当成高风险投资的一小部分，而不是人生全部赌注的人

## 二、这套知识库不适合谁

如果你现在是以下状态，建议先别急着往下走：

- 指望靠 Web3 快速翻身
- 想借钱投资、加杠杆投资
- 连日常生活预算都还没理清楚
- 无法接受高波动和亏损
- 还没有建立“只用输得起的钱参与”的边界

## 三、建议按这个顺序阅读

### 第一步：先建立边界
先看：

1. **《如何投资理财？（实操步骤）》**
2. **《Web3 投资注意事项》**

先确认三件事：
- 你有没有应急金
- 你准备拿多少钱参与
- 你是否接受 Web3 是高风险投资

如果这一步没想清楚，后面的注册和买入都先别急。

### 第二步：了解并注册交易所
再看：

3. **《交易所是什么？新手为什么要先注册交易所》**
4. **《币安注册》**
5. **《易欧注册》**

这一阶段的目标不是交易，而是：
- 知道交易所是干什么的
- 至少注册好一个常用交易所
- 完成身份认证和基础安全设置

### 第三步：完成第一笔小额买入
再看：

6. **《新手第一笔 Web3 资产买什么？》**
7. **《新手如何完成第一笔 BTC / ETH 买入》**

这一阶段的目标不是赚钱，而是：
- 用小金额跑通完整流程
- 知道人民币如何进入交易所
- 知道如何买入 BTC / ETH
- 先完成第一次正确操作

### 第四步：理解钱包和资产保管
再看：

8. **《钱包是什么？什么时候需要开始用钱包》**
9. **《钱包下载》**

这一阶段的目标是：
- 理解钱包不是“赚钱工具”，而是资产管理工具
- 知道什么时候可以继续放在交易所
- 知道什么时候该开始学习钱包

### 第五步：再看工具和补充内容
最后看：

10. **《工具》**
11. **《AI工具》**
12. **《零成本获得比特币》**
13. **《水龙头 APP》**

这部分是增强层，不是主路径。先完成前面几步，再看这些内容，吸收效率会更高。

## 四、给新手的三个提醒

### 1. 先求不出错，再求赚到钱
刚开始最重要的不是收益，而是先跑通流程、建立边界、减少低级错误。

### 2. 第一次一定用小金额
无论注册、买币、转账还是钱包操作，第一次都建议小额尝试，先把流程跑顺。

### 3. Web3 是高风险投资，不是救命稻草
它只能是你整体资产配置中的一小部分，不能替代应急金、保险和正常生活。

## 结尾一句

如果你是第一次接触 Web3，最好的起点不是“今天买什么暴涨”，而是：
**先把边界搞清楚，再用小金额完成第一次正确参与。**
